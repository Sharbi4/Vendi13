# Product Overview

VendiBook is a marketplace platform for food trucks, food trailers, ghost kitchens, vendor lots, and related equipment. The platform supports both rental and sale listings.

## Core Features

- **Listings**: Create and manage rental or sale listings for food service assets
- **Bookings**: Book rentals with calendar availability, add-ons, and instant booking
- **Payments**: Stripe-powered payments with escrow for sales, payouts to hosts
- **Messaging**: Real-time messaging between buyers/renters and hosts/sellers
- **Reviews**: Two-way review system with host responses and sentiment analysis
- **Identity Verification**: Stripe Identity for user verification
- **AI Assistant**: Subscription-based AI chat assistance for users
- **Analytics**: Host dashboards with booking trends, earnings, and performance metrics

## User Roles

- **Guests/Buyers**: Browse listings, make bookings, purchase items
- **Hosts/Sellers**: Create listings, manage availability, receive payouts
- **Admins**: Verify listings, moderate reviews, manage platform

## Business Model

- Platform fee on transactions (10%)
- Featured listing promotions
- AI Assistant subscription ($X/month with 7-day trial)
